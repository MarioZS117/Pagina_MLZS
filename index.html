¿Qué es el método de Gauss-Seidel?

El método de Gauss-Seidel es un algoritmo iterativo utilizado para resolver
sistemas de ecuaciones lineales. Se utiliza principalmente cuando se tiene
un sistema grande de ecuaciones lineales y se busca encontrar las soluciones
de manera numérica.

Este método funciona para ecuaciones que obedezcan la forma que aparece en la siguiente
imagen, esta misma representa una matriz de ecuaciones, en donde:

  • a: Coeficiente de la variable.
  • Subindice nm: Posición del coeficiente.
  • bn: Respresenta los lados derechos.




Ejemplo en Excel

A usos prácticos haremos uso de la herramienta Excel para demostrar
el proceso a seguir en este método.
A continuación resolveremos el sistema de ecuaciones:
2x + 8y -z = 11
-x + y -4z = 3
5x -y + z = 10
Toleracia: 0.001

Hallar Matríz Ideal

Para este caso tenemos que nuestra matriz de coeficientes no cumple
el parámetro de que los números de la diagonal principal sean mayores
que la suma de los valores adyacentes, por ende tenemos que hacer un
ajuste en los renglones para que podamos continuar con el método.

Nota: al hacer los ajustes para un renglon se deben mover también
sus lados derechos.

Iteraciones

Para la iteración 0 se establecen valores de 0, seguidamente
en la iteración 1 se utilizan las funciones antes obtenidas
para obtener los nuevos valores de x,y,z en las distintas funciones.
Teniendo estos nuevos valores e itera de acuerdo al valor de norma
obtenido y a la tolerancia de fallo que tengamos permitida.
